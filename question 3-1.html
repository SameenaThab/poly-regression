Generalization¶

When a machine learning model learns concepts that fit with given set of data(training data) and uses applies them to new data(test data) which are not part of training data refers to Generalization in Machine Learning¶

A good machine learning model should be able to generalize never seen data using what it learned from training data.¶

Underfitting and overfitting are two extreme cases of generalization. A good model must neither underfit nor overfit, it should at a optimal sweet spot between underfitting and overfitting.¶

Overfitting¶

Overfitting happens when model fits the training data too well.This is caused to low bias and high variance between the model and the original data.¶

def norm(x):
    n=x-np.mean(x)
    n=n/np.std(x)
    return n

def polybasis(x,n):
    p=[]
    for i in range(n):
        p.append(x**i)
    return np.asarray(p).reshape([len(x),n])

def error(y,pred):
    m=(pred-y)**2
    return sum(m)/len(y)

def poly_regressorhelp(phi,y,l,n):
    I=np.eye(n)
    i=inv(l*I+np.dot(phi.T,phi))
    w=np.dot(i,np.dot(phi.T,y))
    return w

def poly_regressor(x,y,test,testl,n):
    phi=polybasis(norm(x),n)
    l=0
    w=poly_regressorhelp(phi,y,l,n)
    trainpred=np.dot(phi,w)
    testpred=np.dot(polybasis(norm(test),n),w)
    trainerror=error(y,trainpred)
    testerror=error(testl,testpred)
    print('test error:',testerror)
    print('train error:',trainerror)
    t = x
    a = y
    b = trainpred
    plt.plot(t, a,'g-', label='true label') # plotting t, a separately 
    plt.plot(t, b, 'r-',label='predicted label')
    #red_patch = mpatches.Patch(color='red', label='predicted label')
    #green_patch = mpatches.Patch(color='green', label='true label')
    #plt.legend(handles=[red_patch])
    plt.legend(loc='lower left')
    plt.show()
    if(testerror-trainerror>2):
        print('Overfitting occured')
        l_testerr=[]
        l_trainerr=[]
        lam=[0.01,0.1,1,10,100,500,1000,5000,10000]
        for i in lam:
            w=poly_regressorhelp(phi,y,i,n)
            trainpred=np.dot(phi,w)
            testpred=np.dot(polybasis(norm(test),n),w)
            l_trainerr.append(error(y,trainpred))
            l_testerr.append(error(testl,testpred))
        return [l_testerr,l_trainerr]
